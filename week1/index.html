Master Mobile UX Video

Highlights:

  • Mobile devices dominate the market. You replace them more often. You have more per household.
  • Designing for the smallest device first makes it easy to scale up.
  • Responsive does not equal mobile first.
  • Be there. Be useful. Be quick.
  • Design with reachability in mind. Put important things closer to the bottom. (Like menus.)
  • Mobile design needs to run fast otherwise people won't stick around. Reduce image size and optimize css and js.
  • Only include necessary information. Keep things simple and compact.
  • Stop using pop-ups and pop-overs!!!
