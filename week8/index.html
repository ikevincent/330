Chapter 8: Transforms and Transitions

CSS3 Trasform lets you translate, rotate, scale, and/or skew any element on the page.

You can use more than one function by seperating each of them with a space in the order you want the functions applied.

  • Translate: Move elements left, right, up, or down.
  • Rotate: Rotate an element by any degree.
  • Scale: Change the horizontal and veritical scale of an element.
  • Skew: Skew an element by a certain degree on the x and y axis.

Transitions allow values of CSS properties to change over time. Creating short "animations."

You can set the property (such as transform), duration, timing, and delay.

Keyframes allow you to create the rule that your desired animation will follow.

You tell the keyframe where it will start and how it will finish and call the name of your animation at the start to apply it to your animation.

Canvas vs SVG: From the textbook: In summary, if you need to paint pixels to the screen and have no concerns about the ability to retrieve and modify your shapes, canvas is probably the better choice. If, on the other hand, you need to be able to access and change specific aspects of your graphics, SVG might be more appropriate.

Drag and Drop: You can use this API to specificy elements that can be dragged and dropped on your website.

You designate the draggable elements and then make sure that you also designate where they can be dropped to add interactivity to a website.
